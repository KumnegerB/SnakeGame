# Snake Game (Python + Pygame)

A simple, modern Snake game built with Python and Pygame.

## Run

1. Ensure Python 3.8+ is installed.
2. Dependencies are already installed in the workspace's virtual environment.
3. Launch the game:

```powershell
"D:/School Documents/Programming/Projects/Web Projects/SnakeGame/.venv/Scripts/python.exe" "D:/School Documents/Programming/Projects/Web Projects/SnakeGame/src/snake.py"
```

## Controls

- Arrows / WASD: Move
- P: Pause/Unpause
- Space: Restart (when game over)
- Esc: Quit

## Notes

- The snake wraps at screen edges.
- Speed increases as you score.
